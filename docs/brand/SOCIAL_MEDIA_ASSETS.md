## Overview

This guide provides specifications for all social media assets needed to maintain consistent CloudSync Ultra branding across platforms.

### Design Principles

1. **Consistency** - Same visual language across all platforms
2. **Clarity** - Message readable at a glance
3. **Brand Recognition** - Immediate identification as CloudSync Ultra
4. **Platform Optimization** - Tailored for each platform's requirements

### Color Usage

| Use Case | Colors |
|----------|--------|
| Background | Brand gradient (#6366F1 to #8B5CF6) or Dark (#1A1A2E) |
| Text | White (#FFFFFF) for contrast |
| Accents | Status colors as needed |

## Twitter/X Assets

### Header Image

| Attribute | Value |
|-----------|-------|
| **Dimensions** | 1500 x 500 px |
| **Aspect Ratio** | 3:1 |
| **File Format** | PNG or JPG |
| **Max File Size** | 5 MB |
| **Safe Zone** | Center 1200 x 400 px |

#### Design Specifications

```
+------------------------------------------+
|             UNSAFE ZONE (150px)          |
|  +------------------------------------+  |
|  |                                    |  |
|  |   CloudSync Ultra                  |  |
|  |   "One app. Every cloud."          |  |
|  |                                    |  |
|  |   [Cloud provider icons]           |  |
|  |                                    |  |
|  +------------------------------------+  |
|             UNSAFE ZONE (150px)          |
+------------------------------------------+
```

#### Content Guidelines

- **Primary text:** Centered horizontally
- **Logo:** Optional, if used place in safe zone
- **Tagline:** "One app. Every cloud. Complete control."
- **Visual elements:** Subtle cloud icons, sync indicators
- **Avoid:** Text in outer 150px (cropped on mobile)

#### Layout Recommendations

**Option A: Gradient Background**
- Full brand gradient background
- White text centered
- Provider icons arranged in arc

**Option B: Dark Background**
- Dark (#1A1A2E) background
- Logo centered top
- Key features listed below

### Profile Image

| Attribute | Value |
|-----------|-------|
| **Dimensions** | 400 x 400 px |
| **Display Size** | 200 x 200 px (desktop), smaller on mobile |
| **Cropping** | Circular |
| **File Format** | PNG (transparent) or JPG |
| **Max File Size** | 2 MB |

#### Design Specifications

- Use icon-only logo (no wordmark)
- Center icon in canvas
- Account for circular crop
- 50px padding from edges minimum

## YouTube Assets

### Channel Art (Banner)

| Attribute | Value |
|-----------|-------|
| **Dimensions** | 2560 x 1440 px |
| **File Format** | PNG or JPG |
| **Max File Size** | 6 MB |

#### Safe Zones

YouTube channel art must account for multiple device displays:

| Device | Visible Area |
|--------|--------------|
| TV | 2560 x 1440 px (full) |
| Desktop | 2560 x 423 px (center strip) |
| Tablet | 1855 x 423 px (center) |
| Mobile | 1546 x 423 px (minimum safe) |

#### Design Template

```
+----------------------------------------------------------+
|                                                          |
|                    TV VISIBLE (full)                     |
|    +------------------------------------------------+    |
|    |              DESKTOP VISIBLE                   |    |
|    |    +--------------------------------------+    |    |
|    |    |         TABLET VISIBLE              |    |    |
|    |    |    +----------------------------+   |    |    |
|    |    |    |      MOBILE SAFE AREA      |   |    |    |
|    |    |    |   CloudSync Ultra          |   |    |    |
|    |    |    |   [Logo] [Tagline]         |   |    |    |
|    |    |    +----------------------------+   |    |    |
|    |    |         1546 x 423 px               |    |    |
|    |    +--------------------------------------+    |    |
|    |                 1855 x 423 px                  |    |
|    +------------------------------------------------+    |
|                    2560 x 423 px                         |
+----------------------------------------------------------+
                    2560 x 1440 px
```

#### Content Guidelines

- **Critical content:** Must fit within 1546 x 423 px center
- **Logo:** Centered in mobile safe area
- **Text:** Minimal, large enough to read at 1546px width
- **Background:** Extends to full 2560 x 1440 px

## Open Graph Images

Open Graph images appear when links are shared on social platforms.

### Standard OG Image

| Attribute | Value |
|-----------|-------|
| **Dimensions** | 1200 x 630 px |
| **Aspect Ratio** | 1.91:1 |
| **File Format** | PNG or JPG |
| **Max File Size** | 8 MB (recommended < 1 MB) |

#### Design Specifications

```
+------------------------------------------+
|                                          |
|         CloudSync Ultra                  |
|         ─────────────────                |
|                                          |
|    [App Screenshot or Icon]              |
|                                          |
|    One app. Every cloud.                 |
|    Complete control.                     |
|                                          |
|         cloudsync.app                    |
+------------------------------------------+
        1200 x 630 px
```

#### Content Guidelines

- **Logo:** Top or center, clearly visible
- **Headline:** Large, readable at 400px wide thumbnail
- **Visual:** App screenshot, icon, or illustration
- **URL:** Optional branding at bottom
- **Background:** Brand gradient or dark theme

### OG Image Variants

Create variants for different page types:

| Page Type | Focus | Example |
|-----------|-------|---------|
| Homepage | Brand overview | Logo + tagline + provider icons |
| Features | Specific feature | Feature screenshot + benefit |
| Blog Post | Article preview | Title + category + visual |
| Download | CTA focused | "Download Free" + app icon |

## Content Guidelines

### Visual Style

| Attribute | Guideline |
|-----------|-----------|
| **Colors** | Use brand palette exclusively |
| **Typography** | SF Pro or system fonts |
| **Imagery** | Clean, minimal, tech-forward |
| **Icons** | SF Symbols style, consistent weight |
| **Photos** | High quality, relevant to cloud/sync |

### Messaging Hierarchy

1. **Brand name:** CloudSync Ultra (always visible)
2. **Value proposition:** What we do
3. **Supporting details:** Features, benefits
4. **CTA:** Website, download link

### Text Legibility

| Element | Minimum Size |
|---------|--------------|
| Brand name | 48px at 1200px width |
| Tagline | 24px at 1200px width |
| Body text | 18px at 1200px width |
| Captions | Avoid in social graphics |

### Platform-Specific Tone

| Platform | Tone | Content Focus |
|----------|------|---------------|
| Twitter/X | Conversational, quick | Tips, updates, engagement |
| LinkedIn | Professional, detailed | Business value, case studies |
| YouTube | Educational, helpful | Tutorials, demos, guides |
| Instagram | Visual, aspirational | Lifestyle, behind-scenes |

### Quality Checklist

Before publishing any social asset:

- [ ] Dimensions match platform requirements
- [ ] File size under platform limits
- [ ] Text within safe zones
- [ ] Brand colors accurate (#6366F1, #8B5CF6)
- [ ] Logo properly positioned
- [ ] Text legible at display size
- [ ] Preview on target platform
- [ ] Dark/light mode compatible (if applicable)
