# Architecture Decision Records (ADRs)

This directory contains Architecture Decision Records for CloudSync Ultra.

## What is an ADR?

An ADR captures an important architectural decision along with its context and consequences.

## Template

Use `ADR-TEMPLATE.md` for new decisions.

## Status Definitions

- **Proposed** - Under discussion
- **Accepted** - Decision made
- **Deprecated** - No longer applies
- **Superseded** - Replaced by another ADR
